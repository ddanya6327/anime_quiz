# Anime Quiz

誰でも投稿できるWEBクイズサービス

開発期間： 2020/11 ~ 2020/11（2週）

# Technologies used

- HTML/CSS
- JavaScript (ES6+)
- React
- React Router
- PostCSS
- Firebase
- Cloudinary API (image upload)
